# Changelog

## [1.0.1] - 2026-03-06

### 🚀 **Production Release**
**Полностью рабочее приложение для поиска попутчиков на MongoDB Atlas**

### ✅ **Исправления:**
- **MongoDB Atlas интеграция** - надежная бесплатная база данных
- **Формат данных API** - исправлена совместимость фронтенда и бэкенда
- **Отображение адресов** - теперь показываются корректно
- **Кнопки карт** - восстановлена работа с Яндекс.Картами
- **Система заявок** - полный цикл создания и обработки

### 🎯 **Функции:**
- ✅ **Регистрация/Вход** - JWT аутентификация
- ✅ **Создание поездок** - водители могут создавать поездки
- ✅ **Поиск поездок** - пассажиры могут искать все поездки
- ✅ **Система заявок** - пассажиры подают заявки
- ✅ **Принятие/Отклонение** - водители управляют заявками
- ✅ **История заявок** - пассажиры видят статус своих заявок
- ✅ **Яндекс.Карты** - выбор адресов на интерактивной карте
- ✅ **PWA** - установка на телефон как приложение

### 🔧 **Технический стек:**
- **Backend:** Node.js + Express + MongoDB Atlas
- **Frontend:** Vanilla JavaScript + Яндекс.Карты API
- **Аутентификация:** JWT токены
- **Развертывание:** Render.com (бесплатный тариф)
- **База данных:** MongoDB Atlas (512MB бесплатно)

### 🌐 **Деплой:**
- **Сайт:** https://poputchiki.onrender.com
- **Статус:** Production Ready
- **Мониторинг:** Health check доступен
