# 🌐 Настройка бесплатного домена для Попутчиков

## 🆓 Варианты бесплатных доменов:

### 1. Freenom (рекомендую)
- **Домены:** `.tk`, `.ml`, `.ga`, `.cf`
- **Срок:** 12 месяцев бесплатно
- **Продление:** можно продлевать бесплатно
- **Регистрация:** https://www.freenom.com/

**Инструкция:**
1. Зайти на freenom.com
2. Проверить доступность домена (например: `poputchiki.tk`)
3. Зарегистрировать домен
4. В DNS settings указать IP сервера

### 2. GitHub Pages (простой вариант)
- **Домен:** `poputchiki.github.io`
- **Стоимость:** полностью бесплатно
- **Ограничения:** только статические сайты

### 3. Netlify (современный вариант)
- **Домен:** `poputchiki.netlify.app`
- **Стоимость:** бесплатно
- **Преимущества:** SSL, CDN, автоматический деплой

---

## 🚀 План настройки:

### Шаг 1: Выбор домена
```
Рекомендую: poputchiki.tk
Альтернативы: 
- poputchiki.ml
- poputchiki.ga  
- poputchiki.cf
```

### Шаг 2: Настройка DNS
После регистрации домена нужно настроить:
```
Type: A Record
Name: @ (или poputchiki)
Value: IP_АДРЕС_СЕРВЕРА
TTL: 3600
```

### Шаг 3: Настройка SSL
```
Бесплатный SSL через Let's Encrypt
Автоматическое обновление сертификата
```

- **Домен:** 0 руб/год

- **Итого:** ~300-500 руб/мес

### Netlify вариант:

## 🎯 Рекомендуемый вариант:

### Netlify + GitHub (бесплатно)
1. Загрузить код на GitHub

## 🛠️ Технические требования:

### Для Netlify:
```json
{
  "build": {
    "command": "npm install",
    "publish": "server"
  },
  "redirects": [
    {
      "from": "/*",
      "to": "/index.html",
      "status": 200
    }
  ]
}
```

### Для VPS:
```bash
# Установка Node.js
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# Установка MongoDB
wget -qO - https://www.mongodb.org/static/pgp/server-6.0.asc | sudo apt-key add -
echo "deb [ arch=amd64,arm64 ] https://repo.mongodb.org/apt/ubuntu focal/mongodb-org/6.0 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-6.0.list
sudo apt-get update
sudo apt-get install -y mongodb-org

# Запуск приложения
npm install
npm start
```
