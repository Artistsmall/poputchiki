# 🚗 Попутчики v1.0.1 - Production Release

## 📋 **Обзор**
Полностью функциональное веб-приложение для поиска попутчиков с развертыванием на production.

---

## 🎯 **Основные функции**

### 👤 **Пользователи**
- **Регистрация** с выбором роли (водитель/пассажир)
- **Вход** по email и паролю
- **JWT аутентификация** с токенами
- **Профиль** пользователя

### 🚗 **Для водителей**
- **Создание поездок** с указанием маршрута
- **Выбор адресов** на интерактивной Яндекс.Карте
- **Управление поездками** (просмотр, удаление)
- **Обработка заявок** от пассажиров
- **Принятие/отклонение** заявок

### 🚶 **Для пассажиров**
- **Поиск поездок** по всем доступным маршрутам
- **Подача заявок** на поездки
- **Отслеживание статуса** своих заявок
- **Просмотр деталей** поездок и водителей

---

## 🛠️ **Технический стек**

### Backend
- **Node.js** + **Express.js**
- **MongoDB Atlas** (бесплатный кластер M0)
- **JWT** для аутентификации
- **bcryptjs** для хеширования паролей

### Frontend
- **Vanilla JavaScript** (ES6+)
- **Яндекс.Карты API** для геолокации
- **PWA** для установки на телефон
- **Responsive Design** для всех устройств

### Деплой
- **Render.com** (бесплатный тариф)
- **Автоматический CI/CD** через GitHub
- **SSL сертификат** (HTTPS)
- **Health monitoring**

## 📊 **API Эндпоинты**

### Аутентификация
- `POST /api/auth/register` - регистрация
- `POST /api/auth/login` - вход

### Поездки
- `POST /api/rides` - создать поездку
- `GET /api/rides` - поездки водителя
- `GET /api/rides/search` - поиск всех поездок
- `GET /api/rides/:rideId/requests` - заявки на поездку

### Заявки
- `POST /api/rides/:rideId/requests` - создать заявку
- `GET /api/requests/passenger` - заявки пассажира
- `PUT /api/requests/:requestId/accept` - принять заявку
- `PUT /api/requests/:requestId/reject` - отклонить заявку

### Система
- `GET /api/health` - статус сервиса

NODE_ENV=production
